# 更新日志

## [v20260316] - 2026-03-16 - 蓝图编辑增强与本地工具补充

Hi all, 过去这一轮更新主要聚焦在蓝图编辑体验、属性配置可靠性以及本地工具链补充上。我们为平台新增了天线模型编辑器和属性值校验能力，同时继续完善进程编辑器与属性编辑器细节，并补齐了面向自动化场景的本地工具支持，方便大家在网页端和本地工作流之间更顺畅地协作。

### 亮点：
1. **天线模型编辑器**：蓝图平台新增前端天线模型编辑器，可以在模型库中直接创建、打开和编辑天线模型，支持矩阵编辑与预览展示，便于更高效地完成天线相关建模工作。
2. **属性值校验规则**：属性定义现在支持配置基于 CEL 的校验规则，蓝图前端会在运行时对属性值进行校验，模型编辑器也新增了校验规则帮助说明页面，降低复杂属性配置出错概率。
3. **本地工具链补齐**：新增面向脚本化和自动化场景的 `miaosuan-agent` 非交互式 CLI，同时 `miaosuan-cli` 升级至 `v0.3.2`，支持 `miaosuan run -y` 自动确认上传，更适合批处理和自动化执行。

### 新增：
1. **进程编辑器初始状态自动维护**：创建第一个状态时会自动设置为初始状态；删除当前初始状态后，会自动补选下一个状态作为初始状态，减少手动调整。
2. **属性校验帮助说明**：模型编辑器新增属性值校验规则帮助页面，方便查阅表达式写法与使用方式。

### 优化：
1. **属性编辑器反馈增强**：属性编辑器现在会显示当前命中的常用值名称，便于快速识别当前使用的是哪组预设值。
2. **属性定义编辑体验**：属性表支持调整属性顺序，同时保持默认值编辑器风格一致，属性名称字段也修复了中文输入过程中的失焦问题。
3. **进程编辑器可读性**：优化进程编辑器中状态文字背景显示，提升状态节点文字在复杂画布中的可读性。
4. **菜单与交互细节**：节点右键菜单中的“编辑属性”项被前置，地图右键菜单完成本地化，模型库翻页时会自动清空已选项，减少误操作。
5. **本地命令行体验**：`miaosuan-cli` 增加自动确认上传选项，提升脚本化使用体验。

### 修复：
1. **属性类型切换残留问题**：切换属性类型时会自动重置默认值，并移除不再适用的 `option` 类型配置，避免产生无效属性定义。
2. **属性布尔预设值问题**：修复 MAC 参数中 `bool` / `toggle` 类型预设值导致开关按钮灰色的问题。
3. **窗口位置问题**：限制窗口初始 `y` 坐标，避免弹窗标题栏被屏幕边缘遮挡。
4. **离线地图配置校验**：保存离线地图配置前会校验 URL 模板，减少错误配置被直接写入的情况。

Hi all, 过去的两个周，我们又为妙算平台带来了新的变化，根据用户反馈进行了大量的功能优化和问题修复，同时新增了场景编辑器的历史记录功能，方便大家进行场景编辑操作的撤销和恢复。我们将会持续关注用户反馈，提升平台的易用性和稳定性。

### 新增：
1. **场景编辑器历史记录**：新增场景编辑器历史记录功能，可以撤销和恢复之前的场景编辑操作，支持快捷键和菜单栏操作。

1. **批量导出文件格式**：调整批量导出模型文件格式，使用`.lantu.pkg`作为扩展名，便于与普通压缩包区分。
2. **默认启用所有在线地图**：优化地图配置逻辑，默认启用所有在线地图，并默认使用天地图地形图作为底图，提升用户体验。
3. **升级依赖库**：升级部分前端依赖库版本，提升平台稳定性和性能表现。
4. **模型更新同步**：优化模型更新逻辑库，模型保存后其更新的属性等接口会立即生效，无需重新刷新加载新模型。
5. **进程模型细节优化**：将模型库中的进程模型调整到设备模型后，更符合建模流程逻辑。优化双击逻辑，状态机中转移线的双击操作优化为删除转移线中间点，右键可打开转移线代码。
6. **状态机移动优化**：优化状态机中状态和转移线的移动逻辑，避免带有自环的状态移动时转移线恢复默认形状的问题。
7. **VSCode扩展更新**：上述进程编辑器更新也同步应用到了本地开发工具的VSCode扩展中，提升本地开发体验 <a target="_blank" href="https://github.com/znetlink/miaosuan/blob/master/docs/miaosuan-cli.md#%E4%B8%8B%E8%BD%BD%E5%9C%B0%E5%9D%80">下载地址</a> 、 <a target="_blank" href="https://gitcode.com/znetlink/miaosuan/blob/master/docs/miaosuan-cli.md#%E4%B8%8B%E8%BD%BD%E5%9C%B0%E5%9D%80">国内镜像</a> 。

### 修复：
1. **批量导出样式问题**：修复批量导出时调制解调模型的文本显示问题。
2. **属性配置问题**：修复属性配置中，点击恢复默认值按钮后无法保存的问题。
2. **数组属性配置问题**：修复数组类型属性在属性配置中无法正确编辑的问题。
3. **页面卡顿问题**：修复某些情况下，页面资源占用过高造成的操作卡顿问题，操作更加流畅。
4. **工具栏图标同步删除问题**：修复已经收藏的平台模型被删除时，工具栏图标仍然存在的问题。
5. **工具栏图标同步更新问题**：修复平台模型名称更新时，工具栏图标仍然保持旧名称造成的按钮无效问题。
